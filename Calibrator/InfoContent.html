Konfiguracja USR-N520

1. Podłącz urządzenie do sieci Ethernet i zasilania.
2. Otwórz przeglądarkę i wpisz adres 192.168.0.7
3. Zaloguj się do panelu konfiguracyjnego:
Login: admin
Hasło: admin
4. Ustaw parametry zgodnie z poniższą tabelą:
Setting Socket A Socket B
Working Mode TCP Server TCP Client
Operating Mode 1 0
Local Port Number 26 0
Remote Port Number - 20108
Remote Server Address - IP komputera na którym uruchomiony jest kalibrator w trybie podglądu danych pomiarowych z miernika
Uwaga: Dla Socket B w polu "Remote Server Address" wpisz adres IP komputera, na którym uruchomiona jest aplikacja Calibrator. Adres IP komputera wyświetlany jest w pasku tytułowym aplikacji.
