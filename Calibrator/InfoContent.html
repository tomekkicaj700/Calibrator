Konfiguracja USR-N520

1. Podłącz urządzenie do sieci Ethernet i zasilania.
2. Otwórz przeglądarkę i wpisz adres 192.168.0.7
3. Zaloguj się do panelu konfiguracyjnego:
Login: admin
Hasło: admin
4. Ustaw parametry zgodnie z poniższą tabelą:
Socket A Socket B
Mode TCP Server TCP Client
Local/Remote Port 23 20108
Baud Rate 115200 115200
Data Bits 8 8
Stop Bits 1 1
Parity None None
Flow Control None None
Uwaga: Pozostałe ustawienia pozostaw domyślne. Po zapisaniu zmian zrestartuj urządzenie, jeśli panel o to poprosi.
